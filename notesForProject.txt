Sơ đồ luồng dữ liệu cho cách MakeFile hoạt động trong hệ thống này:

    Giai đoạn 1 (Compile/Assemble): Các file .c và .s đơn lẻ được dịch thành .o (Object files). Đây là lý do có quy tắc mẫu $(BUILD)/%.o: %.c.

    Giai đoạn 2 (Linking): Trình liên kết (ld) gom tất cả file .o lại, kết hợp với file linker.ld (để biết địa chỉ RAM/Flash của con Cortex-M3) nhằm tạo ra file .elf.

    Giai đoạn 3 (Post-process): Dùng objcopy để chuyển file .elf (chứa nhiều thông tin debug) thành file .bin gọn nhẹ để nạp xuống chip.
